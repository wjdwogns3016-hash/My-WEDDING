[전세버스 신청 기능 연결 방법]

1. Google Sheets에서 새 스프레드시트를 하나 만듭니다.
2. 상단 메뉴의 "확장 프로그램 → Apps Script"를 엽니다.
3. google-apps-script.gs의 코드를 전부 붙여넣고 저장합니다.
4. Apps Script 우측 상단의 "배포 → 새 배포"를 선택합니다.
5. 유형은 "웹 앱"으로 선택합니다.
6. 실행 사용자는 "나", 액세스 권한은 "모든 사용자"로 설정합니다.
7. 배포 후 발급되는 URL 중 끝이 /exec 인 주소를 복사합니다.
8. 청첩장 config.js에서 아래 부분에 붙여넣습니다.

   busReservation: {
     ...
     submitUrl: "https://script.google.com/macros/s/........../exec"
   }

9. GitHub Pages에 config.js / index.html / script.js / styles.css를 기존 파일과 교체합니다.
10. 청첩장에서 직접 한 번 테스트 신청한 뒤 Google Sheets의 "전세버스 신청" 탭에 저장되는지 확인합니다.

※ 신청 마감일 문구는 넣지 않았습니다.
※ 배너 문구는 "버스를 이용하시는 분께서는 꼭 작성해 달라"는 내용으로 구성했습니다.
※ 출발 장소와 시간이 확정되면 config.js의 routeInfo에만 적으면 배너에 자동 표시됩니다.


[이번 버전 추가 사항]
- 전세버스 신청 버튼 아래에 '신청 연락처로 추후 개별 일정 안내' 문구가 표시됩니다.
- 신청 완료 메시지에도 연락처로 추후 안내된다는 문구가 표시됩니다.
- 전달받은 Google Apps Script /exec URL이 config.js에 연결되어 있습니다.


[따라다니는 빠른 메뉴 추가]
- 모바일: 오른쪽 아래 '메뉴' 버튼을 누르면 카테고리 목록이 열립니다.
- PC/태블릿: 화면 오른쪽에 카테고리 목록이 고정되어 따라다닙니다.
- 현재 보고 있는 구간이 자동으로 강조됩니다.
- 메뉴 구성: 처음 / 우리의 이야기 / 갤러리 / 오시는 길 / 마음 전하실 곳 / 전세버스
